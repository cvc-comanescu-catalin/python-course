🍎 Python Course — Setting Up Your Windows Development Environment

Welcome! Please install all the tools listed below before our first session.

The whole process takes around 15–20 minutes. If you hit any problems, just reply to this email and we will help you.
 
  📋 Tools Overview
  • Python 3 — the language interpreter
  • Visual Studio Code — lightweight, powerful code editor (free)
  • Python extension for VS Code (by Microsoft) — IntelliSense, debugging, linting, Jupyter support
  • Git for Windows  OPTIONAL  — version control
  • VS Code built-in Git support  OPTIONAL  — source control directly inside the editor once Git is installed
 
   1   Install Python
 

Download the latest stable Python 3 from the official website:

⬇ python.org/downloads

Click the large yellow “Download Python 3.x.x” button — the site detects your OS automatically and offers the correct 64-bit Windows installer.

Installation steps

  1. Run the downloaded  .exe  file.
  2. Before doing anything else, tick “Add python.exe to PATH” at the bottom of the first screen (see image below).
  3. Click Install Now and wait for it to finish.
  4. Click Close.
  ⚠ Critical — Add Python to PATH:
The “Add python.exe to PATH” checkbox is unchecked by default in some versions of the installer. Without it, Python will not be usable from the terminal, Command Prompt, or VS Code. Do not skip it!

Python 3 installer — check “Add python.exe to PATH” before clicking Install Now

Verify the installation

Open Command Prompt ( Win + R  → type  cmd  →  Enter ) and run:

  C:\> python --version
Python 3.13.x

C:\> pip --version
pip 24.x from ...

Seeing version numbers means Python is installed correctly. ✅

 
   2   Install Visual Studio Code
 

VS Code is a free, open-source code editor built by Microsoft. It runs on Windows, macOS, and Linux.

⬇ code.visualstudio.com

Click Download for Windows. Choose the System Installer (recommended over the User Installer).

Installation steps

  1. Run the downloaded  VSCodeSetup-x64-*.exe  file.
  2. Accept the licence agreement and click Next.
  3. Leave the destination folder as default and click Next.
  4. On the “Select Additional Tasks” screen, make sure all checkboxes are ticked — especially:
    ✅ Add “Open with Code” to Windows Explorer context menu
    ✅ Add to PATH (available after restart)
  5. Click Install, wait for it to finish, then click Finish.
  ⚠ Add VS Code to PATH:
Checking “Add to PATH” on the Additional Tasks screen lets you open any folder in VS Code from any terminal window simply by typing  code . . Very useful — keep this ticked!

VS Code installer — ensure “Add to PATH” is checked on the Additional Tasks screen

 
   3   Install the Python Extension for VS Code
 

This official Microsoft extension adds Python IntelliSense (auto-complete), linting, debugging, Jupyter notebook support, variable explorer, and much more to VS Code.

Installation steps

  1. Open Visual Studio Code.
  2. Press  Ctrl + Shift + X  to open the Extensions panel (or click the grid icon in the left sidebar).
  3. Type  Python  in the search box.
  4. Click on “Python” by Microsoft — it will be the top result with 100 M+ installs.
  5. Click the Install button.

Extensions panel — search for “Python” and install the one published by Microsoft

Extension ID:  ms-python.python 

View in VS Code Marketplace
 
   4   Install Git for Windows  OPTIONAL 
 

Git is the world’s most widely used version control system. It’s not strictly required for the Python course itself, but it is an essential tool for any developer and we recommend installing it now.

⬇ git-scm.com/download/win

The download starts automatically when you visit the page (it detects 64-bit Windows).

Installation steps

  1. Run the downloaded installer.
  2. Click Next through the first several screens — the defaults are all fine.
  3. On the “Adjusting your PATH environment” screen, select the recommended option:
    “Git from the command line and also from 3rd-party software”
  4. Continue clicking Next through the remaining screens (defaults are fine), then click Install.
  ⚠ Add Git to PATH:
Choose the recommended option on the PATH screen. This ensures Git works seamlessly inside VS Code’s built-in terminal, PowerShell, and Command Prompt — with zero extra configuration.

Git installer — select the recommended option to add Git to your PATH

Verify the installation

  C:\> git --version
git version 2.x.x.windows.x
 
   5   Use Git in VS Code  OPTIONAL 
 

VS Code already includes built-in Git support, so you do not need to install an extra Git extension for the course.

  1. Make sure Git for Windows is installed first if you want to use version control inside VS Code.
  2. Open Visual Studio Code and click the Source Control icon in the left sidebar, or press  Ctrl + Shift + G .
  3. Open a folder that contains a Git repository, or choose Initialize Repository to start a new one.
  4. Use the Source Control view to review changed files, write commit messages, and commit your work without leaving the editor.
  No additional Git extension is required for this course. Installing Git for Windows is enough to enable VS Code’s built-in Source Control features.
 
  ✅ Final Verification
 

Once everything is installed, open Command Prompt or PowerShell and run the commands below to confirm the tools are on your PATH. The Git check is optional if you skipped Git for Windows:
  C:\> python --version
Python 3.13.x

C:\> pip --version
pip 24.x from ...

C:\> code --version
1.9x.x
...

C:\> git --version
git version 2.x.x.windows.x

All four commands returning version numbers means your development environment is ready to go. 🎉

If a command says “is not recognized as an internal or external command”, the tool was not added to PATH correctly. Try re-running its installer and ticking the PATH option — or reply to this email and we will help you sort it out.
