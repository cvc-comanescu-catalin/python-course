# 🐍 Python for Beginners

A 5-session hands-on Python course for absolute beginners. Each session is **1 hour 40 minutes** and includes a presentation, live coding, and a take-home assignment.

---

## 📋 Prerequisites

- No prior programming experience needed
- A computer with internet access
- Curiosity and willingness to learn!

---

## 📅 Course Schedule

| # | Theme | Duration | Assignment |
|---|-------|----------|------------|
| 1 | Introduction to Python & Programming Basics | 1h 40min | "About Me" Script |
| 2 | Control Flow — Making Decisions & Repeating Actions | 1h 40min | Number Guessing Game |
| 3 | Functions & Modular Thinking | 1h 40min | Calculator with Functions |
| 4 | Data Structures — Lists, Tuples & Dictionaries | 1h 40min | Student Gradebook |
| 5 | File I/O, Error Handling & Mini-Project | 1h 40min | To-Do List App |

## 🚀 Getting Started

### Option 1 — Install Python locally (Recommended)

1. Download Python 3.10+ from [python.org/downloads](https://www.python.org/downloads/)
2. Install **VS Code** from [code.visualstudio.com](https://code.visualstudio.com/) and add the **Python extension**
3. Or install **Thonny** (beginner-friendly IDE) from [thonny.org](https://thonny.org/)
4. Verify your installation: open a terminal and run `python --version`

### Option 2 — No installation needed

- Use **Google Colab** at [colab.research.google.com](https://colab.research.google.com/) — runs Python in your browser for free

## 📚 Session Details

### Session 1: Introduction to Python & Programming Basics

| Time | Topic |
|------|-------|
| 15 min | What is Python? History, use cases |
| 15 min | Setting up Python environment |
| 10 min | First program: Hello World |
| 10 min | Variables & Data Types: `int`, `float`, `str`, `bool` |
| 10 min | Operators: arithmetic, assignment |
| 10 min | User Input with `input()` and type conversion |
| 10 min | Comments and code readability |
| 20 min | Live coding & practice |

### Session 2: Control Flow — Making Decisions & Repeating Actions

| Time | Topic |
|------|-------|
| 10 min | Recap + Q&A |
| 15 min | Comparison Operators |
| 15 min | Conditional Statements: `if`, `elif`, `else` |
| 10 min | Logical Operators: `and`, `or`, `not` |
| 15 min | `while` loops, `break`, `continue` |
| 15 min | `for` loops, `range()`, iterating over strings |
| 20 min | Live coding: FizzBuzz, number guessing game |

### Session 3: Functions & Modular Thinking

| Time | Topic |
|------|-------|
| 10 min | Recap + assignment review |
| 15 min | Why functions? DRY principle |
| 15 min | Defining functions: `def`, parameters, `return` |
| 10 min | Default parameters & keyword arguments |
| 10 min | Scope: local vs global variables |
| 10 min | Built-in functions: `len()`, `abs()`, `round()`, `type()`, `max()`, `min()` |
| 10 min | Importing modules: `math`, `random` |
| 20 min | Practice |

### Session 4: Data Structures — Lists, Tuples & Dictionaries

| Time | Topic |
|------|-------|
| 10 min | Recap + assignment review |
| 20 min | Lists: creation, indexing, slicing, methods |
| 10 min | Looping through lists |
| 10 min | List comprehensions intro |
| 10 min | Tuples |
| 15 min | Dictionaries: key-value pairs, methods |
| 5 min | Nested structures |
| 20 min | Practice |

### Session 5: File I/O, Error Handling & Mini-Project

| Time | Topic |
|------|-------|
| 10 min | Recap + assignment review |
| 15 min | Reading files: `open()`, `read()`, `readlines()`, `with` statement |
| 10 min | Writing files: `write()`, `writelines()`, append mode |
| 15 min | Error handling: `try`, `except`, `finally` |
| 10 min | String formatting: f-strings deep dive |
| 5 min | Course review |
| 35 min | Mini-Project: Command-Line To-Do App guided build |
